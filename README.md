# Maries MP3 Downloader

YouTube-Links reinkopieren, MP3s direkt auf den Player laden!

## Download

**[>>> MariesMP3Downloader.exe herunterladen <<<](MariesMP3Downloader.exe)**

Einfach oben auf die Datei klicken und dann auf "Download".

## So geht's

1. `MariesMP3Downloader.exe` herunterladen und doppelklicken
2. Ordner auf dem MP3-Player waehlen
3. YouTube-Links reinkopieren
4. "Alle Songs herunterladen" klicken - fertig!

Beim allerersten Start wird ffmpeg automatisch heruntergeladen (braucht kurz Internet).
