# Maries MP3 Downloader

YouTube-Links reinkopieren, MP3s direkt auf den Player laden!

## Download

**[>>> Hier klicken zum Herunterladen <<<](https://github.com/Reffn/MariesYoutubeDownloader/releases/latest)**

Beide Dateien runterladen:
- `MariesMP3Downloader.exe`
- `yt-dlp.exe`

## So geht's

1. Beide Dateien in den **gleichen Ordner** legen
2. `MariesMP3Downloader.exe` doppelklicken
3. Ordner auf dem MP3-Player waehlen
4. YouTube-Links reinkopieren
5. "Alle Songs herunterladen" klicken - fertig!

Beim allerersten Start wird ffmpeg automatisch heruntergeladen (braucht kurz Internet).
